  1. Product Quality Check (QC)
    • Visually inspect all products before packaging.
    • Check for scratches, broken parts, or defects.
    • Count and verify all components if it's a multi-piece item.
  2. Battery-Operated Toys
    • Test the toy to ensure it functions.
    • Remove demo batteries and inform the customer if batteries are not included.
  3. Cloth & Plush Toys
    • Ensure clean condition, no loose threads or damage.
    • Fold gently to retain shape before packaging.
  4. Packaging Materials
    • Use bubble wrap or tissue wrap for delicate items.
    • Use zip-lock bags for small toys.
    • Apply branded sticker seals when applicable.
  5. Boxing & Sealing
    • Choose an appropriately sized box ‚Äî not too loose or too tight.
    • Seal with branded or brown packing tape.
    • Include a Thank You card or Mindspark insert (if available).
  6. Label Placement
    • Stick the label flatly on the top, avoiding seams or folds.
    • If handwritten, use a black marker and write clearly.
  7. Label Making Process
    • Login to the Mindspark website dashboard.
    • Navigate to WooCommerce > Export Orders.
    • Select the date range for the orders you want to process.
    • If needed, input the label range to select specific orders.
    • Download the Excel file.
    • Open the file and set the amount to zero for any Bkash or advance-paid entries.
    • Save the updated Excel file.
    • Open this URL: üîó Label Generator (Google Colab)
    • Upload the Excel file and generate the labels.
    • Carefully review all labels to ensure accuracy.
    • Double-check the layout, names, and address details.
    • Print the labels.
  8. Final Checklist Before Dispatch
    • Product cleaned and verified
    • Working test done (if needed)
    • All parts checked
    • Proper packaging applied
    • Label attached
    • Package photo taken (if required)
